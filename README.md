# Python Lead Generation Automation

This project automates startup lead generation using Python by combining CSV dataset processing with basic web scraping techniques. The workflow includes data cleaning, duplicate removal, website and LinkedIn link generation, email generation, and automatic Excel export.

## Features

* Dataset processing
* Data cleaning
* Duplicate removal
* Website generation
* LinkedIn link generation
* Email generation
* Web scraping integration
* Basic automation trigger using Python schedule library
* Excel automation

## Technologies Used

* Python
* Pandas
* Requests
* BeautifulSoup
* OpenPyXL
* Schedule

## Output

The final cleaned lead data is automatically exported into an Excel file (`final_leads.xlsx`).
